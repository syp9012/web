Google icon library

Bootstrap 글리프콘을 사용하려면 HTML head 내부에 다음 줄을 추가하세요 . <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css"> 참고: 다운로드나 설치가 필요하지 않습니다!
